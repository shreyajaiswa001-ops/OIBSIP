# Sales Prediction using Python

## Project Overview

This project predicts product sales based on advertising expenditure using Machine Learning.

## Features Used

* TV Advertising
* Radio Advertising
* Newspaper Advertising

## Target Variable

* Sales

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Machine Learning Algorithm

* Linear Regression

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Model Training
6. Prediction
7. Model Evaluation

## Result

The Linear Regression model achieved an R² Score of 0.8994.

## Conclusion

The model successfully predicted sales based on advertising expenditure and demonstrated strong predictive performance.

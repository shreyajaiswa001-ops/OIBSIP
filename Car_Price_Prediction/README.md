# Car Price Prediction using Machine Learning

## Project Overview

This project predicts the selling price of a car based on various features such as year, present price, fuel type, transmission type, and kilometers driven.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Features Used

* Year
* Present Price
* Driven Kms
* Fuel Type
* Selling Type
* Transmission
* Owner

## Target Variable

* Selling Price

## Machine Learning Algorithm

* Linear Regression

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Encoding
5. Train-Test Split
6. Model Training
7. Prediction
8. Model Evaluation

## Result

R² Score: 0.8467

## Conclusion

The model successfully predicts car selling prices using historical car data with good accuracy.
